1.  Problem Statement 

The aim of this practical is to process raw text data and convert it into a structured format that can be used for machine learning.
Raw text contains unnecessary words, symbols, and variations, so we perform text cleaning, lemmatization, and stop word removal.
After preprocessing, we apply label encoding and convert the text into numerical form using TF-IDF.

2.  Explanation 

In this practical, we first cleaned the text by removing special characters, numbers, and converting all text to lowercase.
Then, we applied lemmatization to reduce words to their base form (e.g., running → run).

Next, we removed stop words such as is, the, and because they do not add important meaning.
After that, we used label encoding to convert categorical labels into numeric values.

Finally, we applied TF-IDF to convert the processed text into numerical vectors. This helps in identifying the importance of words in the dataset.
All the processed outputs were then saved for further use.

3. Conclusion 

This practical demonstrates how raw text data can be cleaned and transformed into meaningful numerical form using NLP techniques.
By applying preprocessing steps and TF-IDF, we make the data suitable for machine learning models and improve analysis accuracy.
